# Paradise Nursery - Plant Shop

A React-based e-commerce application for buying houseplants. Built with Redux for state management and React Router for navigation.

## Features
- Browse plants by category (Succulents, Ferns, Flowering)
- Add plants to shopping cart with one click
- Update quantities or remove items from cart
- Responsive design

## Technologies
- React
- Redux Toolkit
- React Router
- CSS

## Setup
1. Clone the repository
2. Run `npm install`
3. Run `npm start` to launch development server
4. Build for production: `npm run build`
